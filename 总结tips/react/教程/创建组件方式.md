# react组件创建方式

> react定义组件有两种方式 
>
> 下面代码是用create-react-app脚手架生成的项目里的环境下测试
>
> 注意：
>
> 1. 函数名 必须**大写**，就当当做是构造函数
> 2. class  后面的组件名字必须**大写**
> 3. 一定要注意**this**
> 4. 哪些地方能用es6语法？
> 5. **props是别人（父组件）改自己，state是自己改自己**
> 6. 使用class关键字创建的组件（**有状态组件**），有自己的私有数据和生命周期函数，使用function创建的组件（**无状态组件**）只有props没有自己的私有数据和生命周期函数

### 函数方式

```
// 定义元素 传递过来的hello必须解构赋值一下,Message的形参是个对象，对象里面才有msg，所有要解构赋值
/*
1.首字母一定要大写
2.组件定义完以后可以向react元素一样去使用，不需要在render的第一个参数写那些了
*/
let Message = ({msg}) => {
	return <h1>{msg}</h1>
}
/*换一种写法*/
let Message = (props) => {
	return <h1 style={props.style}>{props.msg}---{props.id}</h1>
}
let obj = {
	name: 'x',
	age: '90'
}
// 使用组件
ReactDOM.render(<Message ...obj id={5} msg={"hello"} style={{color:"red"}}/>,document.getElementById('root'))

// 如何理解 怎么一会{}  一会 {{}}
// 一句话总结：x = {y} y是表达式
```



### class 继承方式

```
// Clock 是组件的名称 需要继承React下的一个属性
class Clock extends React.Component {
	// 1.构造器
    constructor () {
    	// Clock继承了React.Component父类，所以在自定义的构造器中，必须调用super()
    	// 只有调用了super() 才能使用this关键字
        super()
        // 状态可以存放组件内部一些变化的值，状态只能内部来改变，外边改变不了，相当于vue中的data() { return {} }
        this.state = {
            time: new Date().toLocaleString()
        }
    }
    // 生命周期函数，挂载后执行一个方法
    componentDidMount () {
        // 每隔1s重新修改状态，新状态更新老状态，重新render渲染，视图更新
        window.setInterval(()=>{
            // setState改变状态
            this.setState({
                time: new Date().toLocaleString()
            })
        },1000)
    }
    // return 只能返回一个react元素，也就是return <h1></h1><h1></h1>，不可以，在最外层包一个div就行
    // 不是粗暴的冲洗渲染，domdiff算法，会处理该更新什么地方，不该更新什么地方
    // 2.核心方法最后渲染dom的  {this.props.msg} 就把传递过来的msg获取到了
    render () {
        return (<h1>{this.state.time}</h1>)
    }
}
// <Clock/> 这里的这个自定义标签，其实就是Clock的一个实例
ReactDOM.render(<Clock msg={'hello'}/>,document.getElementById('root'))
```

### class创建组件方式 props和state

```
// Person组件
// 需要安装这个
import PropTypes from 'prop-types'
/*
* 属性 props
* 状态 state
* 共同点，都可以更新视图
* 不同点，属性是外界传进来更改视图，状态时组件内部进行更改
* 
* props 和vue的v-bind传值类似
* state 和.vue里面的data类似
*/
class Person extends React.Component {
    // 默认属性
    static defaultProps = {
        name: "默认的名字"
    }
    // 对传递过来的数据进行验证是否否和要求
    static propTypes = {
        // eslint-disable-next-line no-undef
        name: PropTypes.string,
        age: PropTypes.number.isRequired,
    }

    constructor() {
        super();
        this.state = {
            // 为组件增加一个初始的状态默认值为true
            happy: false
        }
    }
    // 必须用箭头函数否则还要处理this
    change = () => {
        this.setState({
            happy: !this.state.happy
        })
    }
	// render实例方法  
    render() {
        let heart = this.state.happy ? "开心" : "难过"
        return (
            <div>
                <div className="haha">姓名：{this.props.name}</div>
                <div>心情：{heart}</div>
                {/*点击按钮改变状态，触发函数*/}
                <button onClick={this.change}>改变心情</button>
            </div>

        )
    }
}

// 要求age必填，并且是number如果不写age，或者age={"10"} 控制台都会报错
ReactDOM.render(<Person name={"取代默认"} age={10}/>, document.getElementById('root'))
```
